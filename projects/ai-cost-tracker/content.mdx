## The idea

AI Cost Tracker helps creatives understand what their AI work actually costs. The
first version is a dev-loadable Chrome extension that watches supported AI tools,
estimates usage, and lets users assign each session to a project.

## What works in the MVP

- ChatGPT, Claude, Adobe Firefly, fal.ai, Google Gemini/Flow, Canva AI,
  Midjourney, Runway, and Meta AI page detection.
- Visible conversation or prompt text detection where the page exposes it.
- Estimated input and output tokens based on page text length.
- API-equivalent text-model cost estimates, plus supported fal.ai video model
  estimates from endpoint, duration, resolution, and audio settings.
- Manual cost override for credit, subscription, image, video, and design tools
  without a pricing rule.
- Project tagging, usefulness rating, and short notes.
- Local-only storage through Chrome extension storage.
- JSON export for backup or later analysis.

The estimate is deliberately framed as an estimate. Subscriptions, API usage,
image credits, and per-model inference costs are billed differently, so this is a
practical creative-production ledger, not an official billing dashboard.

## Install from the lab

1. Download the extension zip from **Open project**.
2. Unzip it somewhere stable on your computer.
3. Open Chrome and go to `chrome://extensions`.
4. Turn on **Developer Mode**.
5. Click **Load unpacked**.
6. Select the unzipped `ai-cost-tracker` folder.
7. Pin the extension from Chrome's extensions menu.

## Try it

Open a supported AI page, then open the extension popup. It should detect the
current tool, estimate token usage when visible text is available, and prefill
the log form. Select or create a project, add a rating or note, and log the
entry.
